# Shebam

Web3 payment app

- [**Live app**](http://shebam.w3hc.org)
- [w3pk SDK repo](https://github.com/w3hc/w3pk)

## Install

```bash
pnpm i
```

## Run

Create a `.env` file:

```bash
cp .env.template .env
```

Then:

```bash
pnpm dev
```

## Build

```bash
pnpm build
```
